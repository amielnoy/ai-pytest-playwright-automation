# Session 25 - Exercises

## Exercise 1: Choose The Mechanism

For each workflow, choose skill, plugin, MCP, or slash command:

- Review a test file for flaky locator patterns.
- Open a browser and inspect a failing local page.
- Summarize a GitHub pull request and its CI checks.
- Generate a test file using the same team template every time.

## Exercise 2: Design A QA Skill

Write a short skill outline for reviewing Playwright tests. Include purpose, expected inputs, review criteria, and output format.

## Exercise 3: Plugin Permission Review

Pick one plugin idea and document what it needs access to, what it must never change automatically, and what fallback behavior should exist.

## Exercise 4: MCP Boundary

Describe a browser-debugging task that truly needs Playwright MCP. Then describe a similar task that can be solved by reading code only.

## Exercise 5: Slash Command Spec

Draft a `/generate-api-test` slash command. Include input format, generated files, allowed commands, and review expectations.
