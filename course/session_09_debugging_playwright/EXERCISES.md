# Session 9 - Exercises

## Exercise 1: Break and Diagnose

Break one locator intentionally, run one affected test, and record the traceback and artifact evidence.

## Exercise 2: Classify Failures

Classify three failures as selector, timing, data, environment, or product behavior.

## Exercise 3: Remove a Sleep

Replace a fixed wait with a locator action or assertion that waits for state.

## Exercise 4: Failure Note

Write a failure note with command, evidence, root cause, fix, and verification.
