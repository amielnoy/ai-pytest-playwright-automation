# Session 28 - Exercises

## Exercise 1: Pick The Local Model

Choose a model profile for each task:

- Summarize a short CI failure log.
- Generate edge cases for registration validation.
- Classify a traceback from a page-object failure.
- Review a small diff in a pytest fixture.

## Exercise 2: Write A Safe Prompt

Write an Ollama prompt for generating cart test ideas. Make sure it contains no secrets, accounts, cookies, or customer data.

## Exercise 3: Verify The Output

Take one model suggestion and write the deterministic pytest command you would run before trusting it.

## Exercise 4: Fallback Plan

Describe what you would do if the local model gives vague, wrong, or incomplete output.

## Exercise 5: Privacy Review

Review a hypothetical prompt and mark which parts must be removed before sending it to a local model.
