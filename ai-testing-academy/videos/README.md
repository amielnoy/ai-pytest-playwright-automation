# סרטוני הסבר לאתר AI Testing Academy

שמור כאן קובצי MP4 בשמות הבאים — והם יופיעו אוטומטית בסקשן "🎬 סרטוני הסבר" באתר:

| קובץ | נושא |
|---|---|
| `intro-automation.mp4` | אוטומציית UI עם Playwright (POM, fixtures, הרצת בדיקה) |
| `api-contract-tests.mp4` | בדיקות API ו-Contract דרך שכבת ה-services |
| `devops-allure.mp4` | Docker Compose, דוח Allure ודשבורדים ב-Grafana |
| `ai-agents.mp4` | סוכני AI: ניתוח כשלים, תכנון בדיקות ופקודות Slash |

טיפ להקלטה: הקלטת מסך + מיקרופון (macOS: Cmd+Shift+5, או OBS). Playwright גם מקליט אוטומטית ריצות בדיקה כ-webm — אפשר להמיר ל-MP4 עם `ffmpeg -i in.webm out.mp4`.
